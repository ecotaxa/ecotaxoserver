This import page allows to do mass update of existing taxonomy elements.
To import new elements you must use the taxonomy import.

File format is tsv (tab separated values)
You can use the Full TSV Export feature for template
Only theses colums are used:
  • id : Internal Taxo ID used for lookup
  • parent_id : Internal Taxo ID of parent
  • name
  • taxotype : P : Phylo , M : Morpho
  • taxostatus : A : Active, N : Not validated, D : Deprecated
  • id_source : UniEuk ID
  • source_url : URL documenting the source
  • source_desc : Description of taxon
  • creator_email
  • creation_datetime : Format YYYY-MM-DD HH:MI:SS
  • id_instance : Ecotaxa Instance id (cf instance list)
  • rename_to : Internal Taxo ID of destination for taxon automatic renaming recommanded when deprecating.
Only available colums are updated (you can remove column you doesn't touch), extra columns are ignored (you can use exported format as/is)
Display name are recomputed avec import.
